Kebijakan Privasi

Privasi Anda penting bagi kami. Aplikasi ini tidak mengumpulkan, menyimpan, atau mengunggah data pengguna pribadi atau informasi rahasia lokakarya. Semua catatan keausan alat Anda disimpan dengan aman di perangkat lokal Anda (atau dicadangkan secara bersamaan melalui akun iCloud Anda).

Aplikasi ini tidak memerlukan registrasi, tidak perlu berlangganan, dan berjanji untuk selalu menjadi alat produktivitas murni.

Jika Anda memiliki pertanyaan, silakan hubungi dukungan.
